Dataset Context

What is this dataset?

This dataset contains every crime incident reported to the Los Angeles Police Department (LAPD) between January 2020 and December 2024. Each row represents a single crime report — not necessarily a conviction or confirmed crime — as logged by LAPD officers in the Records Management System (RMS). The data is transcribed from original paper crime reports, so minor typographical inconsistencies may exist.

The dataset is published under the City of Los Angeles Open Data initiative and is refreshed periodically. It covers all 21 LAPD geographic divisions (known as "areas" or "bureaus"), spanning the entire city of Los Angeles.

UCR Classification System (FBI)

Crime codes follow the Uniform Crime Reporting (UCR) standard defined by the FBI, which allows comparisons across U.S. jurisdictions. Under UCR, crimes are divided into two categories: Part 1 (serious crimes tracked nationally: homicide, rape, robbery, aggravated assault, burglary, larceny-theft, motor vehicle theft, arson) and Part 2 (all other less-serious offenses). This classification drives the Part 1-2 field in this dataset.

Important caveats before analysis

Dark figure of crime: This dataset only captures reported crimes. Research consistently shows that many crimes — especially sexual assault, domestic violence, and petty theft — go unreported. Numbers here represent a lower bound on actual crime.

Date lag: DATE OCC (when the crime happened) and Date Rptd (when it was reported) are often different. Identity theft, for example, may be discovered weeks or months after the fact. Analysis of crime trends should use DATE OCC as the canonical date.

Location anonymization: Street addresses are rounded to the nearest hundred block to protect victim privacy. Coordinates (LAT/LON) reflect this block-level approximation, not the exact incident location.

Division Record Number

Description

The unique identifier for each crime report in the LAPD Records Management System. It is assigned at the moment a report is filed and never changes. This is the primary key of the dataset — no two rows share the same DR_NO.

Format: The number encodes metadata: the first 2 digits are the year the report was filed, the next 2 digits are the LAPD area code (01–21), and the remaining digits are a sequential counter. Example: 211507896 = filed in 2021, Area 15 (N Hollywood).

Description

The date on which the crime actually occurred. This is the canonical date for temporal analysis and should be the primary date dimension in any dashboard or model. When the exact date is unknown (e.g., a burglary discovered after days away), LAPD uses the earliest plausible date.

Time of Occurrence

Description

The time at which the crime occurred, stored as a 4-digit integer in 24-hour (military) format without a colon. The value must be parsed: divide by 100 for hours, modulo 100 for minutes. When exact time is unknown, LAPD typically records 1200 (noon) as a placeholder.

Geographic Coordinates (Lat / Lon)

Description

Geographic coordinates (latitude and longitude) of the crime location in WGS 84 decimal degrees (EPSG:4326), the same standard used by GPS systems and web maps. Description

Human-readable name
Modus Operandi Codes

Description

Modus Operandi (Latin: "method of operation") codes describe how the crime was committed. Each record may have multiple space-separated codes. They are LAPD-internal numeric codes capturing tactical details: whether a vehicle was used, if a suspect wore a mask, the type of entry method for a burglary, how a victim was approached, etc. These codes are crucial for linking related crimes (serial offenders often share the same M.O.).

Multi-value field: Must be split on space before analysis. Example: "0416 0334 2004 1822" represents 4 separate M.O. codes. A full Mocode reference table is available from the LAPD.

Human-readable name
Additional Crime Codes (1st–4th charge)

Description

These four fields capture up to four crime codes for incidents involving multiple offenses. Crm Cd 1 is the primary charge (mirrors Crm Cd), while codes 2, 3, and 4 are secondary, tertiary, and quaternary charges. For most crimes, only one charge is recorded. The escalating null rates (0% → 93% → 99.8% → 100%) reflect that multi-charge incidents are uncommon.

Use case: When a robbery also involves assault and weapon charges, all three codes appear here. These fields are valuable for analyzing crime complexity and co-occurrence patterns.

Human-readable name
Investigation / Disposition Status

Description

Indicates the current disposition or resolution status of the criminal case. This is a snapshot at time of data export — statuses can change over time as investigations progress. The dominant status (80% "Invest Cont") reflects the reality that most reported crimes in any major city do not result in an arrest. This metric — the clearance rate — is a standard measure of police effectiveness used nationally.

Clearance rate analysis: Divide (AA + AO + JA + JO) by total cases to compute the clearance rate. Nationally, clearance rates average ~50% for violent crimes and ~20% for property crimes. Compare LAPD's rates by crime type and area against these benchmarks.
